Introduction :

Kronos is an open source Sega Saturn/ST-V emulator written in standard C/C++. Based on Yabause/UoYabause/Yaba Sanshiro, the goal is to improve Sega Saturn Emulation.

Before starting :

To get a full experience, follow these steps :

  • Delete your bkram.bin --> by default, the path is C:\Users\yourname\Appdata\local\kronos\.
  • Delete your old kronos.ini or use the blank one provided with the archive.
  • Old Save/Savestate should not working with Kronos 2.1.3.
Back to Top

OpenGL / Compute Shader video core :

Kronos introduce a new video core : Compute Shader core. Here differences with OpenGL core :

  • OpenGL core : supported graphics cards < OpenGl 4.3.
  • Compute Shader core : supported graphics cards >= OpenGl 4.3
Some graphics issues should not be resolved with OpenGl core (i.e : Sega Rally holes on the road).

Specific Kronos Features :

  • Compute Shader support.
  • Windows 7/10 support.
  • STV game support.
  • Kronos Retroarch support (preliminary).
  • Redump archive support (if you have a saturn game from Redump, it's possible to launch the archive without unzipping it).
  • New video options.
  • Filter support (HQ4X, Bicubic, etc...).
  • New kronos interpreter (less CPU usage).
  • CHD images support.
  • More games was fixed.
  • Removed useless options and old code.

How to launch a STV game ?

To launch a STV game, follow these steps (Windows port) :

  • Create a directory with your STV roms (do not unzip them). Add STV Bios with games.
  • Launch Kronos.exe
  • Click on File --> Settings --> then Cartdridge/Memory panel.
  • In Cartridge, select STV Rom game.
  • Click on ... to add the path to the STV roms. Wait a few seconds. Games should be displayed just above.
  • Below, in Memory, set the path to a .bin file used only for ST-V games book keeping (an ST-V game may behave abnormaly if the .bin file contains the save of its Saturn port).
  • In Input, choose an empty configuration for Port1 and select "ST-V Cabinet". Click on the wrench icon on the right and set each button for player 1.
  • Repeat the same operation for Port 2 for a 2-player game.
  • Click on Ok to close settings window.
  • Push launch icon or in emulation menu, launch.
